I’m a software engineer from Chicago and developer at Enova. I have a passion for user interface design and front end development.

I’m a recent graduate of Rose-Hulman Institute of Technology with a degree in Software Engineering. I love telling stories, taking things apart, and am an avid rock climber.

I’m passionate about one thing in particular… learning. I enjoy doing both UI design and backend development, and staying up to date with the latest web trends and technologies has been a huge goal for me since graduating.

By dabbling in both sides of the web spectrum, I hope to contribute some great software, both visually and functionally, to the open source community in the future. Follow me on Twitter, I’d love to work on a project with you.

Currently, I’m a Ruby on Rails developer specializing in UI development. I’m accustomed to using HAML/ERB and SASS for all of my frontend work, as well as CoffeeScript for any Javascript that is needed.

See how my dev environment is setup here, especially my dotfiles. Or check out some of my code on Github.

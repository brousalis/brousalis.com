google-buttons-sass gem

I’m happy to release google-buttons-sass, a gem that is a SASS-powered version of CSS3 Google Buttons written by Tim O’Donnell.

This gem requires Twitter Bootstrap to be installed, which you can do by importing the CSS files directly to your project or by installing the bootstrap-sass gem.
